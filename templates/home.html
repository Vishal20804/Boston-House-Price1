About The Dataset

The Boston dataset contains several columns that provide information about different aspects of housing in Boston. Some of the important columns include.

1. CRIM:

This column represents the per capita crime rate by town.

2. ZN:

This column represents the proportion of residential land zoned for lots over 25,000 square feet.

3. INDUS:

This column represents the proportion of non-retail business acres per town.

4. CHAS:

This column represents whether the town is bounded by the Charles River (1 if bounded, 0 otherwise).

5. NOX:

This column represents the nitric oxides concentration (parts per 10 million).

6. RM:

This column represents the average number of rooms per dwelling.

7. AGE:

This column represents the proportion of owner-occupied units built before 1940.

8. DIS:

This column represents the weighted distances to five Boston employment centers.

9. RAD:

This column represents the index of accessibility to radial highways.

10. TAX:

This column represents the full-value property tax rate per $10,000.

11. PTRATIO:

This column represents the pupil-teacher ratio by town.

12. B:

This column represents the proportion of Black people by town.

13. LSTAT:

This column represents the percentage of lower status of the population.

14. MEDV:

This column represents the median value of owner-occupied homes
